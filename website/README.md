# Racore Website

This folder contains the complete standalone Racore marketing website.

## Included

- Editorial animated landing page at `/`
- Preserved Racore browser experience at `/browser`
- Responsive desktop and mobile layouts
- Generated transparent 3D Racore intelligence core
- Racore logo and social-sharing artwork
- Local-first system, protocol, safety, provider, workflow, and architecture sections
- Cloudflare Sites-compatible Vinext configuration

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
